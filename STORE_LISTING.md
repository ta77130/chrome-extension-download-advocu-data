## Extension Name
WTM Ambassador Activity Exporter

## Short Description (132 characters max)
Export your Women Techmakers Ambassador activity data from WTM Advocu with a beautiful offline backup including images.

## Detailed Description

Export and backup your Women Techmakers Ambassador activity data from the WTM Advocu platform with ease!

**Perfect for WTM Ambassadors who want to:**
- Create offline backups of their activity history
- Generate beautiful reports of their contributions
- Export data in multiple formats (JSON, Excel, HTML)
- Keep personal records of their WTM journey
- Share their accomplishments with recruiters or colleagues

**Features:**

✅ **Complete Data Export**
- Events organized and attended
- Resources and blog posts created
- Mentorship sessions conducted
- Speaking engagements delivered
- Profile information and skills

✅ **Multiple Export Formats**
- JSON - For developers and data analysis
- Excel (XLSX) - Multi-sheet workbook with organized data
- HTML - Beautiful, interactive visual report
- Complete ZIP backup - Everything including images

✅ **Beautiful Visual Report**
- Professional HTML report with WTM branding
- Your profile with photo and biography
- Activity statistics dashboard
- Organized tabs for each activity type
- Activity cards with images and details
- Works offline - no internet required!

✅ **Privacy & Security**
- All processing happens locally in your browser
- No data sent to external servers
- Complete data privacy
- Open source - verify the code yourself

**How to Use:**

1. Navigate to your WTM Advocu profile page (wtm.advocu.com)
2. Click the extension icon
3. Click "Extract Data" (authentication is automatic)
4. Choose your export format:
   - Download JSON for raw data
   - Download Excel for spreadsheet analysis
   - Download Complete Backup (ZIP) for everything including images and HTML report

**What's Included in the Complete Backup:**

- index.html - Open in any browser for a beautiful visual report
- JSON file - Complete raw data
- Excel file - Organized data in multiple sheets
- images/ folder - All your activity images downloaded locally
- README - Summary and instructions

**Perfect For:**

- Creating portfolios of your WTM contributions
- Backing up your data before losing platform access
- Sharing accomplishments with recruiters
- Personal record keeping
- Annual performance reviews
- Tracking your growth as a WTM Ambassador

**Privacy First:**
This extension only works on wtm.advocu.com and does not collect any personal information. All data processing is local. Read our full privacy policy for details.

**Support:**
For issues or questions, please visit our GitHub repository or contact us.

**Note:** This extension is designed for WTM Ambassadors to export their OWN data. Make sure you have the right to access and download the data before using this tool.

## Justification for Permissions

- **activeTab**: Access the current WTM Advocu page to extract data
- **downloads**: Save exported files to user's computer
- **storage**: Temporarily store authentication tokens for session persistence
- **webRequest**: Capture authentication tokens from API requests to access user's own data
- **wtm.advocu.com**: Access WTM Advocu pages where user is viewing their profile
- **api-wtm.advocu.com**: Fetch activity data from the Advocu API using user's authentication
- **storage.googleapis.com**: Download activity images that are part of user's activities
